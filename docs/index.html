Job Portal

A Full-Stack CRUD Application with Role-Based Access Control

View on GitHub Learn More

Key Features

🔐

Role-Based Access

Three distinct user roles: Job Seeker, Employer, and Administrator with custom permissions

📋

Complete CRUD Operations

Create, Read, Update, and Delete functionality for jobs, users, and applications

💼

Job Management

Employers can post jobs, manage applicants, and track applications in real-time

👥

User Dashboard

Personalized dashboards for job seekers, employers, and administrators

📤

File Upload System

Secure CV and NID document upload with validation and storage

🔒

Secure Authentication

Password hashing with bcryptjs and session management

User Roles & Permissions

👨‍💼 Job Seeker

  • Browse available jobs
  • Apply to job postings
  • Upload CV and NID
  • Track application status
  • View profile and applications

🏢 Employer

  • Post new job listings
  • Edit and delete jobs
  • View applications received
  • Manage applicants
  • Track hiring progress

⚙️ Administrator

  • Manage all users
  • Oversee job postings
  • View all applications
  • System statistics
  • User account management

Database Schema

The application uses a relational MySQL database with the following main tables:

Users Table

Stores user information with role-based identification

Jobs Table

Job listings created by employers with descriptions and requirements

Applications Table

Tracks job applications with status and timestamps

Additional Tables

Supporting tables for file uploads, admin logs, and more

Installation & Setup

1. Clone Repository

git clone https://github.com/Shams-Sajid-Rahman/CRUD-Base-Job-Portal.git

2. Install Dependencies

npm install

3. Configure Database

mysql -u root -p < schema.sql

4. Set Environment Variables

Create .env file with your database credentials

5. Seed Database (Optional)

npm run seed

6. Start Application

npm start
